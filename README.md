# 🌍 TripAdvisor Clone

A responsive TripAdvisor website clone built using HTML and CSS. This project replicates the TripAdvisor UI with clean layout and responsive design.

---

## 🔴 Live Demo

👉 https://pavithra1114.github.io/Tripadvisor-clone/

---

## 🚀 Features

- Responsive navigation bar with logo, search bar, and menu
- Hero section with search functionality
- Travel category tabs (Beach, Hotels, Restaurants, etc.)
- Tour cards section (Coimbatore places)
- Asian countries showcase section
- Travel inspiration banner section
- Footer section
- Fully responsive design for all devices

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Flexbox, Grid, Animation, Media Queries)  
- Font Awesome  
- Google Fonts
